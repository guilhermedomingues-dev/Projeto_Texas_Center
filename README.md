# Projeto Texas Center

Este projeto está sendo desenvolvido por mim em meus tempos lives, para praticar minhas habilidades em Python e JSON.
Sou um grande fã da moda country e, por isso, acabei conhecendo a loja TEXAS CENTER. Me encantei pela loja e decidi criar um projeto inspirado nela.

Abri o site da loja e já me veio o insight de criar um "totem" de auto atendimento para instalar na loja física. Então criei um arquivo JSON com todas as peças que encontrei no site e com todas as informções dos produtos. Criei os arquivos Python e comecei a codar sem parar.

## O que contém no projeto:
1. Os produtos enontrados na loja em um banco de dados JSON.
2. Links que levam para o site oficial da TEXAS CENTER.
3. Cadastro de clientes
4. Visão geral do dono da loja.

O projeto em desenvolvimento.
Versão: 1.1

## Como rodar:
1. Clone o projeto completo e abra no VScode.
2. Abra o arquivo main.py e execute no terminal.
